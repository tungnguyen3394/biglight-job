# SNS 使い方 ガイド画像

Đặt ảnh hướng dẫn cách lấy link SNS vào thư mục này. Giao diện sẽ **tự hiển thị** nếu có file, và tự ẩn nếu chưa có (không cần sửa code).

Tên file mong đợi (PNG/JPG đều được, ưu tiên PNG):

- `facebook.png`  — hướng dẫn lấy link Facebook
- `instagram.png` — hướng dẫn lấy link Instagram
- `tiktok.png`    — hướng dẫn lấy link TikTok

Khuyến nghị: ảnh ngang, rộng ~1000px, dung lượng < 300KB.
